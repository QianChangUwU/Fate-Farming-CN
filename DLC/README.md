# Multi_Zone_Farming_CN（多区域循环刷取）

## 说明
多区域 Fate Farming 配套脚本，需配合 `Fate_Farming_CN.lua` 使用。
本脚本会依次遍历选定的区域列表，在当前区域刷 FATE 直到没有可用的 FATE（通过双色宝石数量是否增加来判断），然后传送到下一个区域。

### 可选区域
| 版本 | 区域 |
| ---- | ---- |
| 7.0 | 奥阔帕恰山、克扎玛乌卡湿地、亚克特尔树海、夏劳尼荒野、遗产之地、活着的记忆 |
| 6.0 | 迷津、萨维奈岛、加雷马、叹息海、天外天垓、厄尔庇斯 |
| 5.0 | 雷克兰德、珂露西亚岛、安穆·艾兰、伊尔美格、拉凯提卡大森林、黑风海 |

### 设置
- **FateMacro**: Fate Farming主脚本在SND中的名字
- **各区域开关**: 勾选要在哪些区域刷FATE（默认仅启用7.0区域）

### 使用方法
1. 确保已安装所有必要插件（Lifestream、vnavmesh、TextAdvance）
2. 在SND中导入主脚本 `Fate_Farming_CN.lua` 和本DLC脚本
3. 在本DLC脚本的设置中填入主脚本的名称
4. 在主脚本设置中开启 `Companion Script Mode`（附属脚本模式）
5. 勾选需要刷取的区域
6. 直接启动本DLC脚本即可

# Atma_Farming_CN（古武魂晶）

## 说明
古武魂晶 Farming 配套脚本，需配合 `Fate_Farming_CN.lua` 使用。
本脚本会依次遍历12个魂晶区域，在每个区域刷 FATE 直到获得足够的魂晶，然后传送到下一个区域。

### 设置
- **FateMacro**: Fate Farming主脚本在SND中的名字
- **NumberToFarm**: 每种魂晶需要刷多少个（默认为1）

### 使用方法
1. 确保已安装所有必要插件（Lifestream、vnavmesh、TextAdvance）
2. 在SND中导入主脚本 `Fate_Farming_CN.lua` 和本DLC脚本
3. 在本DLC脚本的设置中填入主脚本的名称
4. 在主脚本设置中开启 `Companion Script Mode`（附属脚本模式）
5. 直接启动本DLC脚本即可

# HalfSoul_Farming_CN（半魂晶，7.0区域）

## 说明
半魂晶 Farming 配套脚本，需配合 `Fate_Farming_CN.lua` 使用。
本脚本会依次遍历6个7.0区域，在每个区域刷 FATE 直到获得设定数量的半魂晶，然后传送到下一个区域。

### 半魂晶对应区域
| 半魂晶 | 区域 | 物品ID |
| ------ | ---- | ------ |
| 青色半魂晶 | 奥阔帕恰山 | 47744 |
| 碧色半魂晶 | 克扎玛乌卡湿地 | 47745 |
| 绿色半魂晶 | 亚克特尔树海 | 47746 |
| 橙色半魂晶 | 夏劳尼荒野 | 47747 |
| 紫色半魂晶 | 遗产之地 | 47748 |
| 黄色半魂晶 | 活着的记忆 | 47749 |

### 设置
- **FateMacro**: Fate Farming主脚本在SND中的名字
- **每种半魂晶**: 设置需要刷取的数量（0=不刷，1-3个，默认3）

### 使用方法
1. 确保已安装所有必要插件（Lifestream、vnavmesh、TextAdvance）
2. 在SND中导入主脚本 `Fate_Farming_CN.lua` 和本DLC脚本
3. 在本DLC脚本的设置中填入主脚本的名称
4. 在主脚本设置中开启 `Companion Script Mode`（附属脚本模式）
5. 设置每种半魂晶的目标数量（0-3）
6. 直接启动本DLC脚本即可
